Introducción: El Salto a un Nivel Superior

¡Bienvenidos a 3º de la ESO! Ya sois unos expertos en la programación con bloques. Ahora, es momento de dar el siguiente paso y aprender a programar con un lenguaje de código real, como Python. Veréis que muchos conceptos son los mismos, solo cambia la forma en que los escribimos.

Esta unidad es un puente entre lo que ya sabéis y lo que estáis a punto de descubrir.

1. Lenguajes de Programación: Tipos y Paradigmas

Existen miles de lenguajes de programación. Se pueden clasificar de varias formas. La que más nos interesa es la diferencia entre:

Visuales (Bloques)

Como Scratch, son ideales para principiantes. Se basan en arrastrar y soltar bloques que representan instrucciones.

Ventajas:

  • Intuitivos y visuales.
  • Evitan errores de sintaxis.
  • Aprendizaje rápido.

Textuales (Código)

Como Python, son los lenguajes profesionales. Las instrucciones se escriben con texto.

Ventajas:

  • Más flexibles y potentes.
  • Se usan para crear aplicaciones complejas.
  • Mayor control sobre el programa.

2. Elementos Fundamentales de la Programación

Los elementos básicos de un programa son los mismos en Scratch y en Python. La diferencia es cómo se escriben.

Variables: Datos en Contenedores

Una variable es un espacio de memoria para guardar datos (números, texto, etc.).

En Scratch:

`crear variable [puntos]`

En Python:

`puntos = 0`

Estructuras de Control

Son las instrucciones que controlan el flujo del programa. Las principales son las condicionales y los bucles.

En Scratch:

`si entonces { ... }`

En Python (condicional):

`if tocado_raton:`
` # ... código ...`

En Python (bucle):

`while True:`
` # ... código ...`

3. Algoritmos y Lógica con Código

La lógica de un programa sigue siendo la misma. La única diferencia es que ahora la escribimos.

Algoritmos

Un algoritmo es una secuencia de instrucciones para resolver un problema. En Python, la secuencia es la clave: las instrucciones se ejecutan de arriba a abajo.

Ejemplo de secuencia en Python:

`print("Hola")`
`print("Adiós")`

Bucles `for` y `while`

Python nos da más control sobre los bucles. El bucle `for` es ideal para repetir un número exacto de veces, y el bucle `while` para repetir mientras una condición sea verdadera.

Bucle `for`:

`for i in range(5):`
` print("Hola " + str(i))`

Bucle `while`:

`contador = 0`
`while contador < 5:`
` print("Hola")`
` contador = contador + 1`

4. Interacción en el Texto

La interacción en Python se hace con la consola. Podemos pedir datos al usuario (input) y mostrarle información (output).

Actividad Práctica: Crea un programa en Python que pida al usuario su nombre y luego le dé la bienvenida.

`nombre = input("¿Cuál es tu nombre? ")`
`print("Hola, " + nombre)`

Proyecto Final: El Conversor Interactivo

Tu misión es crear dos programas, uno en Scratch y otro en Python, que hagan lo mismo: convertir una temperatura de Celsius a Fahrenheit.

Requisitos Mínimos:

  • Programa en Scratch:
    - Pide al usuario la temperatura en Celsius.
    - Usa una variable para guardar el resultado.
    - Muestra la temperatura convertida en la pantalla.
  • Programa en Python:
    - Pide al usuario la temperatura en Celsius (usando `input()`).
    - Realiza el cálculo matemático `(temperatura * 9/5) + 32`.
    - Muestra la temperatura convertida (usando `print()`).

Evaluación:

Este proyecto se evaluará en base a la funcionalidad de ambos programas, la correcta aplicación de los conceptos (variables, operaciones matemáticas, entrada/salida) y la comparación de cómo se resuelven los mismos problemas con dos lenguajes de programación diferentes.
